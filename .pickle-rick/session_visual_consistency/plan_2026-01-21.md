# Status Indicator Overhaul Implementation Plan

## Overview
Reduce visual noise by minimizing green status dots and aligning indicators with the "God Mode" minimalist aesthetic. Fix logic bugs in connection status.

## Current State Analysis
- `ConnectionStatus.kt:137`: `InlineConnectionStatus` shows 'NO SIGNAL' when `isConnected` is true.
- `ModuleCard.kt:143`: `ModuleRowItem` always displays a `StatusDot` regardless of state.
- `TerminalBadge.kt:162`: `StatusDot` is a basic circle with no glow or depth.

## Implementation Approach
1.  **Refine `StatusDot`**: Add a subtle glow effect using a layered `Box` approach (shadow circle + solid circle).
2.  **Logic Fix**: Flip the text logic in `InlineConnectionStatus`.
3.  **Dashboard Cleanup**: Modify `ModuleRowItem` to hide the dot when the item is in its 'Normal' state (Enabled and Connected).

```kotlin
@Composable
fun StatusDot(
    color: Color,
    modifier: Modifier = Modifier,
    size: Dp = TerminalSpacing.statusDotSize,
    showGlow: Boolean = true
) {
    Box(
        modifier = modifier.size(size),
        contentAlignment = Alignment.Center
    ) {
        if (showGlow) {
            // Subtle glow/outer ring
            Box(
                modifier = Modifier
                    .size(size + 2.dp)
                    .background(color.copy(alpha = 0.2f), CircleShape)
            )
        }
        // Core dot
        Box(
            modifier = Modifier
                .size(size)
                .background(color, CircleShape)
        )
    }
}
```

```kotlin
@Composable
fun InlineConnectionStatus(
    isConnected: Boolean,
    serverName: String? = null,
    modifier: Modifier = Modifier
) {
    Row(
        modifier = modifier,
        verticalAlignment = Alignment.CenterVertically,
        horizontalArrangement = Arrangement.spacedBy(TerminalSpacing.xs)
    ) {
        if (!isConnected) {
            StatusDot(color = TerminalColors.statusOffline)
            Text(
                text = "NO SIGNAL",
                color = TerminalColors.statusOffline,
                style = TerminalTypography.labelSmall
            )
        } else {
            // Minimalist: show text, but maybe no dot or a very subtle one
            Text(
                text = (serverName?.uppercase() ?: "CONNECTED"),
                color = TerminalColors.textSecondary,
                style = TerminalTypography.labelSmall
            )
        }
    }
}
```
